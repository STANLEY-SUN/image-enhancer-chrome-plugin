# 图片画质增强 Chrome 插件 - 设计规格

## 概述

一个 Chrome 扩展插件，支持拖拽本地图片或右键选中网页图片，通过浏览器端 AI 模型（ONNX + WebAssembly）进行超分辨率增强，提供 2x 和 4x 放大选项，并支持滑动对比和手动下载。

## 核心功能

| 功能 | 描述 |
|------|------|
| 本地拖拽增强 | 拖拽图片到插件窗口，自动增强并显示结果 |
| 右键菜单增强 | 网页图片右键 → 增强 → 弹出窗口处理 |
| 2x / 4x 放大 | 用户可选 2 倍或 4 倍放大 |
| 滑动对比 | 拖动滑块实时对比原图与增强后效果 |
| 手动下载 | 增强完成后用户自行选择保存路径 |

## 技术方案

**AI 模型**：Real-ESRGAN 转换为 ONNX 格式，通过 WebAssembly 在浏览器端运行

- 无需 API 密钥，零成本
- 图片不上传，隐私安全
- 离线可用

**文件大小**：约 10-20MB（模型权重）

**限制**：浏览器算力有限，建议 2x 效果最佳，4x 可用但较慢

## 用户交互流程

```
拖入图片 → 选择 2x/4x → 显示处理进度 → 滑动对比预览 → 下载保存
```

### 步骤 1：空状态
- 拖拽区域居中突出显示
- 显示图标 + "将图片拖拽到此处" + "或点击选择文件"
- 悬停有反馈（边框高亮）

### 步骤 2：图片加载
- 显示原图预览 + 文件名 + 尺寸
- 出现 2x 增强 / 4x 增强按钮
- 其他元素（对比视图、下载）隐藏

### 步骤 3：处理中
- 显示加载动画（进度条）
- 显示 "AI 增强处理中..." 文字
- 按钮禁用

### 步骤 4：增强完成
- 滑动对比视图（原图 vs 增强后）
- 显示原尺寸 vs 增强后尺寸
- 显示"下载"和"新图片"按钮

## UI 设计

### 布局
- 最大宽度 480px，整体居中
- 紧凑布局，元素间距合理

### 配色
- 跟随系统自动切换浅色/深色模式
- 主色调：蓝紫渐变 (#4060cc → #6040bb)
- 圆角统一 12px

### 字体
- 现代无衬线字体（如系统默认或 Inter）
- 标题加粗，正文常规

### 交互反馈
- 悬停状态：边框高亮、背景色变化
- 处理中：进度条动画
- 按钮：圆角矩形，统一风格
